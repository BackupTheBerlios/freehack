Die Benutzung von CList

CList ist von std::vector abgeleitet und hat deshalb auch genau die selben Eigenschaften und Funktionen. Man kann CList genauso wie std::vector benutzen, allerdings wurden auch neue Funktionen hinzugefügt, die allerdings nur dazu dienen, den Aufruf der Vektorklasse passender zu den restlichen Funktionen zu gestalten.

Eine einfache Liste

Eine Neue Liste legt man mit 
 CList<type> liste;
an. type muss durch den Variablentyp ersetzt werden, von dem die Elemente der Liste sein sollen. Dann kann man mit 
 liste.Add(object);
Objekte (vom Typ type) in die List aufnehmen.
Der Aufruf der einzelnen Listenelemente erfolgt wie bei einem Array mit 
 liste[elementnummer];
elementnummer geht dabei von 0 bis liste.Count()-1. 
 liste.Count();
gibt die Anzahl der Elemente, die in der Liste gespeichert sind zurück. Die Nummer des größen Elements ist folglich liste.Count()-1, da das erste Element die Nummer 0 hat.

Eine Liste aus Zeigern

Eine Liste kann natürlich auch aus Zeigern bestehen. In diesem Fall kann man der Liste auch NULL hinzufügen. D.h. das Objekt, das man der Liste hinzufügen will muss noch nicht existieren und kann erst im nachhinein erstellt werden. Außerdem muss bei komplexen Objekten nicht das ganze Objekt kopiert werden, da in der Liste ja nur ein Zeiger auf das Objekt gespeichert wird. 
 myobject object1, *object2;
 CList<myobject*> liste;
 // Nullzeiger hinzufügen
 liste.Add(NULL);
 liste[liste.Count()-1] = new myobject;
 //Zeiger auf Objekt hinzufügen
 liste.Add(&object1);
 liste.Add(object2);
